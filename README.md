

 ## GitHub User Finder 🔍

A simple web application that allows you to search for GitHub users and view their profile information and repositories.

# # Features
Search for any GitHub user by username

Display user profile (avatar, name, bio)

Show followers, following, and repository count

List the user's most recent repositories with links

Error handling for users not found


 ## Technologies Used
HTML5

CSS3

JavaScript (Vanilla)

Axios - HTTP client for API requests

Font Awesome - Icons

GitHub REST API - User data
